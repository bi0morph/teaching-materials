Что такое JavaScript?

JavaScript — прототипно-ориентированный сценарный язык программирования. Является реализацией языка ECMAScript. ECMAScript — это встраиваемый расширяемый не имеющий средств ввода-вывода язык программирования, используемый в качестве основы для построения других скриптовых языков. Диалекты: JavaScript, JScript, ActionScript, JScript .NET, QtScript

Среда выполнения кода

JavaScript может выполняться не только в браузере, а где угодно, нужна лишь специальная программа — интерпретатор.

Типы (types)

Простые типы

  • string
  • number
  • boolean
  • null
  • undefined
Когда вы взаимодействуете с простым типом, вы взаимодействуете непосредственно с его значением в памяти. 

var foo = 1,
    bar = foo;

bar = 9;

console.log(foo, bar); // => 1, 9. foo не изменился

Сложные типы

  • object
  • array
  • function
Когда вы взаимодействуете со сложным типом, вы взаимодействуете с ссылкой на его значение в памяти. 

var foo = [1, 2],
    bar = foo;

bar[0] = 9; // bar теперь ссылается на foo

console.log(foo[0], bar[0]); // => 9, 9.

Асинхронные скрипты: defer/async

</script>

Разница между async и defer: атрибут defer сохраняет относительную последовательность скриптов. defer всегда ждёт, пока весь HTML-документ будет готов.
