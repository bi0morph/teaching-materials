Задание: Объекты

Создать объект который будет отображать Вас(осбоенности, действия)

  • Создать пустой объект
  • Добавить свойства со значениями. Это может быть рост, пол, профессия, любимая книга, ширина, высота, цвет. Не обязательно использовать свойства присущие только человеку.
  • Добавить методы(функция сохраненная в свойство). Фунция может выполнять любое действие(лог, арерт, write)
  • Бонус: создайте еще несколько объектов с которыми вы взаимодействуете ежидневно (например, гамбургер)
  • Бонус: сделайте так что бы методы вашего объекта взаимодействовал с этими объектами. (например, получал или менял свойство)
