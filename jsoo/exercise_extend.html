Упражнение: Наследование объектов



В этом упражнении вы будете использовать объектно-ориентированные концепции программирования для наследования объектов в JavaScript.


  1. Используйте repl.it для упражнения, выберите "JavaScript" в списке языков.
  2. Начните ваше решение используяю код из предыдущего упражнения (или скопируйте от сюда)
  3. Создайте новый класс MusicVideo который расширяет Video. Этот конструктор должен также принимать artist в качестве параметра.
  4. Создайте новую сущность класса MusicVideo и вызовите метод watch() у него.
  5. Добавьте метод для MusicVideo, который называется rockOut() и используяю console.log выведите строку вида "Вы зажигали под 'La Bamba' от Ritchie Valens!".
  6. Бонус: Используйте массив данных и цикл for для инициализации 5 объектов MusicVideo.
  7. Бонус: Создайте массив видео данных с нормальным видео и музыкальным видео, переберите эти данные, и создайте объекты для этих данных решая какой именно конструтор использовать Video или MusicVideo.
