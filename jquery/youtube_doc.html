youtube.js документация

Википедия: "Библиотека JavaScript — сборник классов и/или функций на языке JavaScript."

jQuery это пример популярной библиотеки на сегодняшний. youtube.js это другой пример библиотеки, написанный разработчиком Pamela Fox, которая содержит JavaScript методы для вашего практического задания.

В библиотеке youtube.js вы можете использовать всего четыре функции:

  1. Получить идентификатор видео из Video URL

    Синтаксис: 
    youtube.getIdFromUrl(videoUrlOrID)
    Ожидаеться в качестве аргумента (videoUrlOrID) youtube ссылка или идентификатор видео. И метод возвращает идентификатор видео.

    Пример: 
    var vid_id = youtube.getIdFromUrl('http://www.youtube.com/watch?v=epUk3T2Kfno');
//vid_id равен 'epUk3T2Kfno' (обратите внимание где ID в оригинальном url)

  2. Сгенерировать ссылку на миниатюру ссылки на видео

    Синтаксис: 
    youtube.generateThumbnailUrl(videoUrlOrID)
    Ожидаеться в качестве аргумента (videoUrlOrID) youtube ссылка или идентификатор видео. И метод возвращает ссылку на миниатюру для этого видео.

    Пример: 
    var vid_thumbnail = youtube.generateThumbnailUrl('http://www.youtube.com/watch?v=epUk3T2Kfno');
//vid_thumbnail равен 'http://i3.ytimg.com/vi/epUk3T2Kfno/default.jpg' (обратите внимание, что идентификатор видео вставлен в средину ссылки на миниатюру)

  3. Сгенерировать ссылку для просмотра видео по идентификатору видео.

    Синтаксис: 
    youtube.generateWatchUrl(videoUrlOrID)
    Ожидаеться в качестве аргумента (videoUrlOrID) youtube ссылка или идентификатор видео. Возвращает ссылку на видео.

    Пример: 
    var vid_watch = youtube.generateWatchUrl('epUk3T2Kfno');
//vid_watch равен 'https://www.youtube.com/watch?v=epUk3T2Kfno' (обратите внимание где идентификатор в ссылке)

  4. Сгенерировать ссылку для подключения видео по идентификатору

    Синтаксис: 
    youtube.generateEmbedUrl(videoUrlOrID)
    Ожидаеться в качестве аргумента (videoUrlOrID) youtube ссылка или идентификатор видео. Возвращает ссылку для подключения видео.

    Пример: 
    var vid_embed = youtube.generateEmbedUrl('http://www.youtube.com/watch?v=epUk3T2Kfno');
//vid_embed равен 'http://www.youtube.com/embed/epUk3T2Kfno' (обратите внимание где идентификатор в ссылке)
