Что такое библиотека?

Коллекция многоразовых методов для конкретной цели.

Библиотека math может иметь такие функции, как:

math.sum(array);
math.pow(num, num);
math.factorial(num);

Что такое библиотека?

  • Подключение скрипта библиотеки на вашей странице

jQuery: Почему?

Без библиотеки:

var p = document.createElement('p');
p.appendChild(document.createTextNode('Welcome!'));
p.style.cssFloat = 'left';
p.style.backgroundColor = 'red';
p.className = 'special';
document.querySelector('div.header').appendChild(p);

jQuery:

var newP = $('<p>Welcome!</p>');
newP.css({'float': 'left', 'background-color': 'red'});
newP.addClass('special');
$('div.header').append(newP);
короче, кроссбраузерно

jQuery: Сохранение элементов

Если у нас такой html... Yahoo! 

<a id="yahoo" href="http://www.yahoo.com" style="font-size:20px">Yahoo!</a>

Мы можем найти его... 

$('a#yahoo');

Мы можем сохранить его... 

var myLink = $('a#yahoo');

...И мы можем сделать много чего с этим: 

myLink.html();
 → 
'Yahoo!'
 
myLink.attr('href');
 → 
'http://www.yahoo.com'
 
myLink.css('font-size');
 → 
'20px'
 
коротко, кроссбраузерно

jQuery: Изменение элемента


Если мы начнем с такого html: 
<a href="http://www.google.com">Google</a>

Можем использовать jQuery таким образом: 
$('a').html('Yahoo!');
$('a').attr('href', 'http://www.yahoo.com');
$('a').css({'color': 'purple'});

И получим это: 
<a href="http://www.yahoo.com" style="color:purple">Yahoo</a>

jQuery: Создание и хранение


Передайте любую строку html и jQuery создаст его и вернет как коллекцию.


С этим jQuery... Мы создаем этот HTML: 
$('<p>');
 → 
<p></p>
 
$('<p>Welcome!</p>');
 → 
<p>Welcome!</p>
 
$('<p class="intro">Welcome!</p>');
 → 
<p class="intro">Welcome!</p>

Точно так же как с DOM API, мы можем хранить ссылку к нашему новому элементу в памяти...

var myParagraph = $('<p class="intro">Welcome!</p>');

jQuery: изменение


Теперь у нас есть сохраненная ссылка, мы можем вносить дальнейшие изменения в наш элемент.

var myParagraph = $('<p class="intro">Welcome!</p>');
myParagraph.css('font-size','4em');

jQuery: Вставка


Теперь, мы можем взять сохраненную ссылку myParagraph и вставить ее куда-нибудь!

$('body').append(myParagraph);
$('body').prepend(myParagraph);

обычные DOM nodes и jQuery объектам


var paragraphs = $('p'); // массив, jQuery коллекция

var myParagraph = paragraphs[0]; // обычный DOM node

var $myParagraph = $(paragraphs[0]); // объект jQuery

Мы можем пройтись по этому массиву...

for(var i = 0; i < paragraphs.length; i++) {
   var element = paragraphs[i];
   var paragraph = $(element);
   paragraph.html(paragraph.html() + ' wowee!!!!');
};
