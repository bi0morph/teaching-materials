Задания

  1. Создать форму для отправки запроса на поиск. - тесктовое поле - кнопка
  2. События. Отправляется запрос к api: - при нажатии Enter в текстовом поле - при нажатии на кнопку
  1. Создать объект который будет AJAX запросы к api. - делаем запрос по этому адресу http://api.nestoria.co.uk/api?country=uk&pretty=1&action=search_listings&encoding=json&listing_type=buy&page=1&number_of_results=6&place_name=leeds - place_name - это поисковый запрос, который вводиться в тесктовом поле - number_of_results - это количество результатов на странице - page - это номер страницы Пример: https://github.com/bi0morph/propertyBackbone/blob/gh-pages/javascripts/nestoria-api.js
  1. В ответ приходит json (например, переменная data). Важные свойства: - data.request - то что вы запрашивали - data.response.application_response_code и data.response.application_response_text - ответ от сервера о результате поиска application_response_code: 100, 101, 110 - что-то было найдено 200, 202 - поисковое слово сильно обобщено, соответствие не найдено, но есть похожие, возвращает список для уточнения Другие - произошла ошибка
  1. В случае если application_response_code один из (100, 101, 110) нужно вывести рузультат в виде списка с названием, картинками и ценой.
  2. В случае если application_response_code один из (200, 202) нужно вывести рузультат в виде списка с названием
  3. В других случаях с application_response_code. Сообщаем об ошибке.
  4. Так же если ответ от api не пришел в течении 5 секунд, сообщаем об ошибке.
