Командная строка

Оболочка (Shell)

Интерпритатор текстовых команд.

Самая расспрастраненная оболочка это "bash" (Unix / Linux).

Для Windows рекомендуется использовать "Git Bash".

На Mac оболочку можно использовать через приложение "Terminal".

Подсказка оболочки

Подсказка обычно показывает имя текущего пользователя и имя компьютера:

username@hostname:

Чтобы увидеть в какой директории вы находитесь, используйте команду pwd: 

username@hostname: pwd
/Users/username/Desktop

Путь к файлам

Последовательность вложенных каталогов(папок) с файлом или каталогом в конце, разделенные через "/":

  • Относительный: documents/fun/microsoft_jokes.html
  • Абсолютный: /usr/bin/local
  • Корень: /

Сокращения:

  • Текущий каталог: .
  • Родительский каталог: ..
  • Домашний каталог: ~

Изменение каталога

Используйте команду cd для того чтобы переместится в другой каталог:

cd mydirectory/

cd ../

Содержимое каталога

Используйте команду ls чтобы увидеть список подкаталогов и файлов в каталоге:

username@hostname: ls
mydir/
myfile.txt

Чтобы увидеть детали используйте ключ -l, и чтобы увидеть скрытые файлы используйте ключ -a: 

username@hostname: ls -al
-rw-r--r--    16 pamelafox  staff      544 19 Mar 11:40 .dropbox
drwxr-xr-x    16 pamelafox  staff      544 19 Mar 11:40 mydir/
-rw-r--r--    16 pamelafox  staff      544 19 Mar 11:40 myfile.txt

Имена фалов

  • чувствительны к регистру (за исключением Windows)
  • все символы (кроме "/")
  • расширения файлов для удобства пользователя

Пример: 

README   .bashrc   index.html    index.html.old    Bugs List.txt

Создание и удаление каталогов

Чтобы создать каталог используйте команду mkdir:

mkdir myproject

Чтобы удалить пустой каталог используйте команду rmdir:

rmdir myproject

Чтобы удалить не пустой каталог используйте команду rm с ключем : -r

rm -r myproject

Файловая структура

Зависит от системы, но есть расправстраненные каталоги:

  • /bin: Основные команды
  • /etc: Конфигурационные файлы системы
  • /tmp: Временные файлы
  • /usr: Обычные инструменты пользователя
  • /usr/local: Установленные программы пользователя
  • /var: Данные используемые системой

Работаем с файлами

Чтобы создать файл - команда touch:

touch newfile.txt

Копировать файл - cp: 

cp newfile.txt newerfile.txt

Переместить файл - mv: 

mv newerfile.txt betterfilename.txt

Удалить файл - rm: 

rm betterfilename.txt

Редактируем файлы

Можно использовать различные редакторы встроенные в bash:

  • vi myfile.txt
  • emacs myfile.txt
  • pico myfile.txt

Или (в Mac) можно открыть через обычно приложение:

open -a TextEdit myfile.txt

Или (в Window):

notepad myfile.txt
subl myfile.txt

Просмотр файлов

Можно посмотроеть содержимое файла в редакторе, а можно сразу в консоли:

  • cat myfile.txt
  • more myfile.txt
  • less myfile.txt

Можно посмотрет сразу несколько файлов

 cat myfile.txt myfile2.txt myfile3.txt

Поиск в файлах

Испольуйте команду grep для посика файлов:

Найти фразу "hello" в конкретном файле:

grep hello myfile.txt

Найти фразу "hello" во всех текстовых фалах *.txt:

grep hello *.txt

Поиск файлов

Используйте команду find чтобы найти файлы в соответствии с правилом название/metadata:

Найти все текстовые файлы в данном каталоге:

find . -name '*.txt' -print

Перенаправление вывода

По умолчанию, ввод приходит с экрана и вывод идет на экран.

Вы можете использовать < и > чтобы перенаправить ввод/вывод.

Сохранить результат поиска в файл:

grep hello *.txt > results.txt

Использование клавиатуры

  • Стрелки влево/вправо позволяют редактировать команду
  • Стрелки вверх/вниз позволяют выбирать предыдущие команды
  • Клавиша tab автозаполняет имена файлов (auto-completes) во время ввода

История ввода

Используйте команду history чтобы увидеть историю команд:

  564  grep 'jquery' *.txt > results.txt
  565  vi results.txt
  566* open -a TextEdit myfile.txt
  568  history

Также полезно:

  • !!: Последная команда
  • !568: Команда по номеру #
  • !open: Команда по совпадению строки

Время упражнений!

  • Перейдите к вашей папке "Documents" или туда где вы обычно сохраняете проекты.
  • Создайте новую папку с именем "myproject".
  • Перейдите в эту папку.
  • Создайте новый файл с именем "index.html".
  • Измените этот файл так что бы он содержал в себе базовую html страницу с title.
  • Скопируйте файл с другого проекта в эту папку, переименуйте его, и подключеите его в index.html.
  • Откройте этот файл в Chrome и проверье что он работает.
