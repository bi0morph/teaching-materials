Упражнение: Анимация

Кошка которая гуляет сама по себе

  • Начните со страницы с одним тегом img который содержит анимированную GIF шагающей кошки. 
    <!DOCTYPE html>
<html>
 <head>
  <meta charset="utf-8" />
  <title>Шагаюшая кошка</title>
 </head>
 <body>
  
  <img style="position:absolute;" src="http://www.anniemation.com/clip_art/images/cat-walk.gif">
  
 </body>
</html>
  • Добавьте тег script в низу страницы.
  • Создайте переменную и сохраните в нее ссылку на img.
  • Измените стиль тега img так чтобы свойство "left" было "0px", это задаст старт анимации с левой части экрана.
  • Напишите функцию catWalk() которая передвигает кошку на 10px в право, изменяя свойство "left".
  • Вызывайте эту функцию каждые 50 миллисекунд. Ваша кошка должна двигатся слева на право черерз весь экран. 



    var img = document.getElementsByTagName('img')[0];
img.style.left = '0px';
function catWalk() {
  img.style.left = (parseInt(img.style.left) + 10) + 'px';
}
window.setInterval(catWalk, 50);
  • Бонус #1: Когда кошка достигает правого края окна обновите значние left ("0px"). В итоге кошка будет вечно двигаться с лева на право. 



    var img = document.getElementsByTagName('img')[0];
img.style.left = '0px';
function catWalk() {
  var currentLeft = parseInt(img.style.left);
  img.style.left = (currentLeft + 10) + 'px';
  if (currentLeft > (window.innerWidth-img.width)) {
    img.style.left = '0px';
  }
}
window.setInterval(catWalk, 50);
  • Бонус #2: Когда кошка достигает правого края окна, заставьте кошку двигатся обратно - задом на перед. Теперь она должна двигатся вперед и назад - Вечно!. 



    var img = document.getElementsByTagName('img')[0];
img.style.left = '0px';
var walkForwards = true;
function catWalk() {
  var currentLeft = parseInt(img.style.left);
  
  if (walkForwards && (currentLeft > (window.innerWidth-img.width))) {
    walkForwards = false;
  }
  if (!walkForwards && (currentLeft <= 0)) {
    walkForwards = true;
  }
  
  if (walkForwards) {
    img.style.left = (currentLeft + 10) + 'px';
  } else {
    img.style.left = (currentLeft - 10) + 'px';
  }
}
window.setInterval(catWalk, 50);
  • Бонус #3: Когда кошка достигает середины экрана, замените картинку на картинку с танцующей кошкой, заставьте ее танцевать 10 секунд на месте, а затем вернить шагающую кошку и пусть продолжит движение.
  • Бонус #4: Используйте `setTimeout` вместо `setInterval`. Тут понадобится вспомнить про рекурсию.
  • Бонус #5: Переделайте решение через `requestAnimationFrame`.
