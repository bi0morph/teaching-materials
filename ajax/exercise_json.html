Упражнение: JSON

Используя json данные со страницы jsonplaceholder/users создайте список пользователей в html

  • Скопируйте данные со страницы и сохрание текст в виде строки в переменную
  • Используя JSON.parse() преобразуйте строку в массив или объект
  • С помощью document.createElement() и .appendChild() выведите этот список в body
